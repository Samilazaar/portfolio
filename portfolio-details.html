LunetteSITE

LunetteSite est un site vitrine de vente lunette qui permet a l'utilisateur de commander plusieurs gamme de lunettes differentes

Fonctionnement

Connexion

Nouvel utilisateur

L’utilisateur créé tout d’abord son compte :

  • Email ou numéro de téléphone
  • Mot de passe
  • Centre d’intérêts

    • Une liste de choix multiples (ex : politique, sport, philosophie, poésie, cinéma, …)

    Ces éléments pourront ensuite être modifiés dans le profil de l’utilisateur

    Utilisateur existant

    Renseigner le mail ou téléphone et le mot de passe avec la possibilité de cliquer sur « mot de passe oublié » pour changer son mot de passe via un mail envoyé sur l’adresse mail de l’utilisateur ou par sms.

    Modification des informations du compte

    L’utilisateur, une fois connecté, peux à tout moment modifier ses infos

  • Mot de passe
  • Adresse postale
  • Centres d’intérêts

    • Emprunts de livre

    Après s'être connecté, l'utilisateur a la possibilité d'emprunter des livres. Cependant, un délai de deux semaines est nécessaire avant que l'utilisateur puisse commencer à emprunter des livres. Cette période d'attente peut être utilisée pour vérifier l'authenticité des informations fournies ou pour d'autres raisons administratives.

    Fonction administrateur

  • L'application comporte également un compte administrateur avec des privilèges spéciaux. L'administrateur peut :

    Ajouter de nouveaux livres à la bibliothèque, Modifier les informations des livres existants, Supprimer des livres de la bibliothèque. Ces fonctionnalités permettent à l'administrateur de gérer efficacement le catalogue de la bibliothèque et de s'assurer que les utilisateurs ont accès à une collection de livres à jour et pertinente.

    • Partie technique

    L'application a été développée en utilisant les technologies MySQL et Java : MySQL : Utilisé pour la gestion de la base de données. Toutes les informations concernant les utilisateurs, les livres, et les transactions d'emprunt sont stockées dans une base de données MySQL. Cela permet de gérer efficacement les données et de les récupérer rapidement. Java : Utilisé pour développer la logique applicative. Le backend de l'application est écrit en Java, ce qui assure une performance robuste et une intégration fluide avec la base de données MySQL. Java permet également de créer une interface utilisateur interactive et réactive pour une meilleure expérience utilisateur. Grâce à l'utilisation de MySQL et Java, l'application est capable de gérer efficacement les opérations de la bibliothèque tout en offrant une expérience utilisateur fiable et performante.
